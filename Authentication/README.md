
# Authentication System in Python

An command line auth system in Python




## Installation

Install this with git

```bash
  git clone https://github.com/fieryant/python.git
  cd Authentication
```
required modules
```
  sqlite3
  bcrypt
  getpass
```
## Usage/Examples

```javascript
  python authentication.py
```
